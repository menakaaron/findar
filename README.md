# **Findar** - **2026 WINFO Hackathon: Best Product Track**

By: Harshita Keerthipati, Thanishkaa Saravanane, Menaka Aron, Tejaswi Erattu

# **What is Findar?**

Findar is an integratable software/app that helps people find things they’ve lost without needing to tag, scan, or even remember. It passively remembers where objects were last seen using a camera and AR. Instead of scrambling around the house to find anything, users can simply ask Findar where an item was last seen, and it will tell them when and where, guiding them back to that spot. Findar is designed to be plugged/integrated into systems people already use in their homes (Alexa, Ring Security Cameras, etc).

# **How Does It Works?**

Findar connects multiple 360° cameras using LiDAR, YOLO-based object detection, and spatial anchors to build a temporal-spatial map of the home. This creates a real-time 3D understanding of the space that updates as the locations of items (and people) change. The system runs quietly in the background without user interaction. We used Xcode and Swift to implement the native iOS application.
