# **Findar** - **2026 WINFO Hackathon: Best Product Track**

By: Harshita Keerthipati, Thanishkaa Saravanane, Menaka Aron, Tejaswi Erattu

# **What is Findar?**

Findar is an app/software that helps people find things they have lost without needing to tag, scan, or even remember. It passively remembers where objects were seen last using a camera and AR. Instead of scrambling all over the house, you can ask Findar where the object was last seen, and it will tell you when and where, guiding you to that spot.

# **How It Works?**

Findar connects multiple 360° cameras that use LiDAR, YOLO-based detection, and spatial anchors to create a temporal-spatial map of the house. This stores a real-time 3D view of the house on a backend Cloud server that updates as locations of items (and people) move. It runs quietly in the background real-time. We used X-Code and Swift to implement the native iOS application.
